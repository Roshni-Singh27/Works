Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project allows two players to play the classic Tic Tac Toe game in a web browser with a clean and responsive user interface.

🎮 Features
Two-player gameplay (X and O)
Interactive game board
Winner detection
Draw detection
Restart/New Game button
Responsive design
Simple and user-friendly interface
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)

🚀 How to Run
Download or clone the repository:
git clone https://github.com/your-username/tic-tac-toe.git
Navigate to the project folder:
cd tic-tac-toe
Open index.html in your browser.

No additional installation or dependencies are required.

🎯 How to Play
The game starts with Player X.
Players take turns clicking on empty cells.
The first player to get three marks in a row (horizontal, vertical, or diagonal) wins.
If all cells are filled and no player wins, the game ends in a draw.
Click the Restart button to start a new game.
🏆 Winning Conditions

The following combinations result in a win:

[0,1,2]
[3,4,5]
[6,7,8]
[0,3,6]
[1,4,7]
[2,5,8]
[0,4,8]
[2,4,6]
